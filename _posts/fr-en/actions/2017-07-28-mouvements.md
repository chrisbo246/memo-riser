## Mouvements

hocher la tête
: nod

se pencher sur, s'appuyer sur
: lean on

se lever
: get up

coup d’œil
: peek, glance

lancer un coup d'œil
: dart a glance

verser
: pour

renverser
: turn upside down

bise
: cheek kiss

sourire
: grin


## Expressions

tendre l'oreille
: lend an ear

se boucher les oreilles
: clog your ears

fourrer, enfoncer
: shove
*Ex: Fourrer ton doigt dans ton nez.*

coup de tête
: head-butt
